# [Procedural Launcher](https://ronnie-reagan.github.io/procedural_launcher/)

This is a simple game; one meant to be relaxing.

Take your time, enjoy the music and try to get a higher streak than your friends.

Persistent/Saved game data to preserve progress - pick up and play right where you left off

Mobile and Desktop friendly

## Desktop Users:

Due to lack of a windows system or desire to use react and such; youre gonna have to install as a PWA like i do on the chromebook I developed the game on

This basically means that there is no linux or windows binaries; you must play as a web app

### Install for/with Chrome:

- Open your browser
- Navigate to `https://ronnie-reagan.github.io/procedural_launcher/`
- **Click** on the three dots on the top right
- **Hover** over 'Cast, Save and Share'
- **Click** on 'Install page as app...`

## Mobile Users:

Due to lack of caring for mobile specific development this game can be installed as a PWA(Progressive Web App)

This basically means that you cannot install it from the app store in the traditional sense, rather you add it to your homescreen via your browser of choice

### Install for Chrome:

- Open your browser
- Navigate to `https://ronnie-reagan.github.io/procedural_launcher/`
- Tap on 'share' icon (in address bar at top)
- Tap on _More_
- Select 'Add to Home Screen'

### Install for Safari:

- Open your browser
- Navigate to `https://ronnie-reagan.github.io/procedural_launcher/`
- Tap on the three 'dots' to the right of the address bar
- Tap on _Share_
- Tap on _More_ (three dots again)
- Select 'Add to Home Screen'
